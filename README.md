# Graphic-image-editor

Project on computer graphics subject

## Authors
  -Krystian Tokarz
  -Artur Wicha 

## Description

This program was written in C language, uses ALLEGRO library

This program allows :
 * load bitmap
 * using Homogeneous Blur
 * using Gauss Blur
 * using Sharpening bitmap
 * editin bitmap
 * clear the changes
 * save changed bitmap
